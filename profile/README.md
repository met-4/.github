### 스터디진행 주기
- 1주, but 상황에 따라 2주도 가능(1주 1주 휴일 등도가능)
- 1시간~2시간 (ex : 15분 세션 x 2회 / 질의응답 등)
 
### 방법
- 리딩발표자 주도로 진행
Ex) 책을하나 정해서, 돌아가며 챕터를 맡아서발표 및기술 공유/진행
 
### 스케쥴
4-1) 평일겹치지 않는날에 진행
- 평일변수에 의하여복잡해질 수있다.
4-2) 주말온라인으로 진행
